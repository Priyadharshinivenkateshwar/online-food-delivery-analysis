# online food delivery analysis

this project analyzes online food delivery data to understand customer behavior,
delivery performance, and revenue trends.

## project objective
- analyze customer ordering patterns
- identify delivery and operational issues
- evaluate restaurant and cuisine performance
- support business decision-making using data

## tools used
- python
- mysql
- power bi

## project flow
dataset → data cleaning → eda → feature engineering → sql analysis → power bi dashboard

## contents
- data cleaning and preprocessing using python
- exploratory data analysis (eda)
- feature engineering
- sql-based business analysis
- power bi dashboard visualization
